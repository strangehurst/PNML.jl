Unclaimed Labels

XML tags that are not 'claimed' are recursively parsed into a AnyXmlNode tree whose leaf nodes are strings by unclaimed_label.

See AnyElement, anyelement, PnmlLabel, Term.

AnyElement

Main use case if for ToolInfo content. The specification allows any well-formed XML. Only the intended tool needs to understand the content.

TODO Implement a ToolInfo for PNML.jl extensions.

PnmlLabel

Applies label semantics to a vector of AnyXmlNodes. Used for not-yet-implemented labels. Many of the labels used for high-level many-sorted algebra have not been implemented.

See rate for a use case.

Sorts

Figure 11 of the primer lists built-in sorts and functions for Symmetric Nets, a restricted High-Level Petri Net Graph. See SymmetricNet, HLPNG and AbstractHLCore

Some of the restrictions:

  • ArbitrarySort and 'Unparsed' not allowed.
  • carrier sets of all basic (is this built-in?) sorts are finite
  • sorttype of a place must not be a multiset sort (but multiset is in the UML diagram)

What HLPNG adds:

  • declarations for sorts and functions
  • ArbitraryDeclarations: ArbitrarySort, ArbitraryOperator, Unparsed
  • Integers
